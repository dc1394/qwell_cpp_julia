★これは何？
　無限に深い井戸型ポテンシャルのシュレーディンガー方程式を、ヤコビの楕円関数を
　用いて展開するコードです。C++版のビルドには、以下のライブラリが必要です（同梱
　）。
　・Boost C++ Libraries
　・Eigen
